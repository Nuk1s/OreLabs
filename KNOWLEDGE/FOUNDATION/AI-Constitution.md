# AI Constitution

## Purpose

Rules for AI assistants working on OreLabs.

## Rules

1. **Follow existing architecture** — Never invent new systems
2. **Read documentation first** — Understand before implementing
3. **Preserve approved decisions** — Never replace with opinions
4. **Maintain consistency** — Match existing code style
5. **Document changes** — Update documentation after implementation

## Forbidden Practices

- Ignoring repository documentation
- Bypassing architecture
- Duplicating systems
- Silently modifying behavior
- Leaving documentation outdated
- Skipping self-review
- Marking unfinished work as completed

## AI Workflow

1. Synchronize with repository
2. Understand current state
3. Implement only requested scope
4. Self-verify against standards
5. Update documentation
6. Report completion
