# Philosophy

## Purpose

Engineering philosophy for OreLabs.

## Core Beliefs

- Readable code > clever code
- Predictable code > short code
- Maintainable code > fast-written code
- Every script should look like one developer wrote it

## Engineering Values

- Consistency over preference
- Simplicity over complexity
- Explicit over implicit
- Long-term over short-term
