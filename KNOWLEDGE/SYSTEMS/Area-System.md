# Area System

## Purpose

Area gates and world progression.

## Architecture

- **AreaService** — Server area logic
- **AreaHandler** — ProximityPrompt interactions
- **GameConfig.Areas** — Area definitions
- **Profile.Data.CurrentArea** — Current location
- **Profile.Data.UnlockedAreas** — Unlocked areas

## Requirements

Each area requires:
- Equipment Tier X
- Backpack Tier X
